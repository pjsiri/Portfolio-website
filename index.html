My Portfolio

AI Chess Program (my favourite one)

This Chess Game utilises minimax algorithm with alpha-beta pruning to provide a strategic chess experience. With the embedded database, users can save and load their chess games, allowing them to revist and analyse historic moves.

Maze Program

Introducing my maze program! This program is designed to load mazes from text files and employs breadth-first search algorithm to navigate through the maze, finding the optimal path from starting node to exit node.

Mobile Virus Simulator

Welcome to my mobile phone virus transmission simulator, a exploration of mobile virus spread, a virus is randomly initiated on one phone and spreads. Only one phone can repaired a time. I have implemented multi-threading and synchronisation to prevent race conditions.

Noise Remover Program

Introducing my noise removing application for images, a tool that can enhance the visual quality of your pictures! This application utilises the efficient median filtering technique, implemented using the quicksort algorithm, to effectively remove noise from images.
